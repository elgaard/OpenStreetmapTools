Elgaard DK restaurant and food instection report tools

These tools use the Danish food inspection reports (the "smiley" reports) which has a name, a postion and a unique id (fvst:navnelbnr) for all danish restaurants, cafes, canteens, etc.

The tool have links to the food inspection reports and to JOSM Remote Control actions (also works in Merkaartor/git)

  • Food places that is in FVST but not in OSM
    The Remote Control link creates the node in OSM, at the "fvst:navnelbnr" and also "amenity", "cuisine", etc based on the FVST name.
  • Food places that is in both OSM and FVST but missing fvst:navnelbnr id in OSM
    List both the FVST and the OSM name. Draws a red line between the FVST and OSM location - the OSM end is marked with a blue dot.
    The Remote Control link add the fvst:navnelbnr to the OSM node. But please verify that the OSM name and position is correct and edit it in JOSM if not.
  • Foodplaces with wrong locations in FVST
  • Restaurants witout FVST tag
  • Restaurants,cafes, fast_food witout FVST tag
